# Coinbase Clone (Public Deploy Repo)

A React + Tailwind CSS Coinbase clone built with Vite.

## Live Deployment

This repository is intended for public hosting (Netlify) because the classroom repository is private.

## Tech Stack

- React
- React Router
- Tailwind CSS
- Vite

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- SPA redirects for Netlify are configured in `public/_redirects`.
- Footer links are currently non-functional placeholders by design.
